named-regexp

Named capture groups for Java 5/6 regular expressions

Download this project as a .zip file Download this project as a tar.gz file

about!

named-regexp is a thin wrapper for good ol' java.util.regex but with support for named capture groups in Java 5/6. (yes, Java 7 already does this...)

snippet.

The construct for named groups is the same as in Java 7: (?<name>...), as shown below.

NamedPattern p = NamedPattern.compile("abc(?<foo>\\d+)xyz");
NamedMatcher m = p.matcher("abc123xyz");
if (m.find()) {
    System.out.println("found '" + m.group("foo") + "'"); // prints "found: '123'"
} else {
    System.out.println("nothing found...sorry :[");
}
